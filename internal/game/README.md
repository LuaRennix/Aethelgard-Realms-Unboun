# Game Module Structure

Этот модуль содержит основную логику игры и был разделен на несколько файлов для лучшей организации кода:

## Файлы

- `constants.go` - определение констант (размеры экрана, состояния игры, языки)
- `structures.go` - определение структур данных (MenuItem, Game)
- `init.go` - функция инициализации NewGame и вспомогательные функции инициализации
- `update.go` - основная игровая логика обновления состояния (функция Update)
- `draw.go` - функции отрисовки и визуальные элементы (функция Draw и связанные с ней)
- `localization.go` - функции локализации и перевода текстов
- `audio.go` - функции работы с аудио (загрузка, воспроизведение, управление громкостью)
- `handlers.go` - обработчики событий и вспомогательные функции
- `fonts.go` - данные шрифта (генерируется автоматически)
- `game.go` - основной файл пакета с минимальным импортом

## Архитектура

Игра использует Ebiten - 2D игровой движок для Go. Она состоит из трех основных состояний:
- Главное меню (MenuState)
- Игровой процесс (GameState)
- Настройки (SettingsState)

Каждый файл отвечает за свою конкретную часть функциональности, что позволяет легче поддерживать и развивать код.
